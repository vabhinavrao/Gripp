## Technical Recording Notes

### Screen Recording Setup
- **Resolution:** 1920x1080 or 1280x720
- **Frame Rate:** 30 FPS minimum
- **Format:** MP4 (H.264)
- **Audio:** Optional (can be text-only)

### Browser Setup
- Chrome with extension installed
- Zoom level: 100%
- Clear cookies/history for clean demo
- Use light browser theme (neutral background)

### X Account Setup
- Use demo account or public articles
- Prepare 2-3 sample articles:
  1. Long article (500+ words) - shows green badge
  2. Short tweet (20-30 words) - shows blue badge
  3. Tweet with multi media (3+ images/videos)

### Recording Tips
1. **Smooth Mouse Movement** - No jerky movements
2. **Pause Between Actions** - 1-2 seconds for clarity
3. **Highlight Cursor** - Use yellow circle highlight
4. **Text Overlays** - Add in post-production
5. **Transitions** - Smooth fade/dissolve between scenes

### Post-Production
- Add text overlays at specified times
- Include subtle background music (optional)
- Add zoom effects for small UI elements
- Export with fade-in/fade-out
